# Exercícios de Programação Orientada a Objetos (POO)

Este repositório contém exercícios de Programação Orientada a Objetos (POO) do curso C# Essencial Macoratti. Os exercícios foram desenvolvidos para praticar conceitos de POO, como herança, polimorfismo e encapsulamento.
